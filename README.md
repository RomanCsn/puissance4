# Puissance 4

Un jeu de Puissance 4 développé avec React + TypeScript + Vite

## Stack Technique

- React avec TypeScript
- Vite pour le build
- Prisma pour la base de données
- Tailwind CSS pour le styling

## Développement

```bash
npm install
npm run dev
```

## Contribuer

Ce projet utilise les outils suivants pour maintenir la qualité du code :
- ESLint pour le linting
- TypeScript pour la vérification de types
- Vite pour un développement rapide avec HMR
